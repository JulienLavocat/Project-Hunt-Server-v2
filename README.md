# Project-Hunt-Server

## Introduction

Project Hunt is a multiplayer FPS where every 30 seconds you got a new target. During this time interval, each kills rewards :
 - Target kill:  5 pts
 - Hunter kill:  3 pts
 - Regular kill: 1 pts

You do not loose any points on death.
